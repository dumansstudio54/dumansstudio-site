DumansStudio · Site Sahibi Rehberi

Sitenizi yayınlama ve güncelleme rehberi

Bu rehber yalnızca size özeldir; ziyaretçilere açık değildir. Siteniz artık tek bir dosyadan oluşuyor: index.html. Tüm fotoğraflar, yazılar ve tasarım bu dosyanın içine gömülüdür — ayrı klasör veya görsel dosyası yoktur. Bu sayede yüklerken hiçbir şey eksik kalmaz, “site açılmıyor” sorunu yaşanmaz.

Neden tek dosya? Daha önce site; ayrı CSS ve gorseller klasörüne bağlıydı. Bu klasörler sunucuya tam yüklenmeyince site bozuk açılıyordu. Artık her şey index.html içinde olduğu için yüklemesi kusursuz: sadece bu tek dosyayı yüklemeniz yeterli.

1. Yayınlama — GitHub + Netlify

Dosya GitHub'da durur, Netlify GitHub'a bağlanır. GitHub'daki dosyayı her değiştirdiğinizde site kendiliğinden güncellenir. Hepsi ücretsizdir.

1

GitHub deposu oluşturun

github.com/signup → ücretsiz hesap. Sağ üstten New repository → isim dumansstudio-site → Public → Create repository.
2

Dosyayı yükleyin

Depo sayfasında “uploading an existing file” bağlantısına tıklayın. Sadece index.html dosyasını bu pencereye sürükleyin → Commit changes. (İsterseniz README.md'yi de ekleyebilirsiniz; REHBER.html'i yüklemeyin — o size özeldir.)
3

Netlify'ı bağlayın

app.netlify.com → GitHub ile giriş → Add new site → Import an existing project → GitHub → dumansstudio-site deposunu seçin → Deploy. Birkaç saniyede site ...netlify.app adresinde canlı olur. Bu adrese girip çalıştığını görün.
4

dumansstudio.com'u bağlayın (Namecheap)

Netlify'da Domain settings → Add a domain → dumansstudio.com. Netlify size kayıtları gösterir. Namecheap → Domain List → Manage → Advanced DNS:
  • A Record · Host @ · Değer 75.2.60.5
  • CNAME · Host www · Değer SİTENİZ.netlify.app
(Netlify ekranındaki güncel değerleri esas alın; takılırsanız ekran görüntüsü gönderin.)
5

Panel alt adresleri

Namecheap Advanced DNS'te, panel sunucunuza işaret eden iki kayıt:
  • A/CNAME · Host ogrenci · Değer: panel sunucunuzun adresi
  • A/CNAME · Host yonetici · Değer: panel sunucunuzun adresi
(Bu adresi panelinizi barındıran hosting/cPanel hesabınızdan alırsınız.)
GitHub'la uğraşmak istemezseniz (en hızlısı): app.netlify.com/drop sayfasını açın ve index.html dosyasını doğrudan bu sayfaya sürükleyin. Tek dosya olduğu için saniyeler içinde, eksiksiz yayınlanır. Güncellemek istediğinizde yeni index.html'i tekrar sürüklersiniz.

2. Fotoğraf veya yazı değiştirmek

Fotoğraflar ve yazılar artık index.html dosyasının içine gömülü olduğundan, bunları doğrudan kendiniz tek tek değiştirmek teknik bilgi ister. En kolay ve garantili yol:

Bana söyleyin, sizin için güncelleyeyim. Hangi fotoğrafı değiştirmek istediğinizi (“en üstteki arka plan”, “Merve Hoca görseli”, “galeri 3. foto” gibi) belirtip yeni fotoğrafı gönderin; ya da değişecek yazıyı yazın. Size güncellenmiş yeni index.html dosyasını veririm — siz de onu GitHub'a (veya Netlify'a) yükleyip yayınlarsınız. 2 dakikalık iş.

Sitede şu an değiştirilebilecek başlıca görseller: açılış logosu, üstteki sinematik arka plan kareleri, Hizmetler (Reformer & EMS) görselleri, Hakkımızda (Merve Hoca) görseli, Instagram galerisi (6 foto), Otopark Kartı ve ON Dijital logosu.

Sık güncelleme yapacaksanız: Size özel, “Fotoğraf Seç → Yükle → Kaydet” şeklinde basit bir yönetim paneli (CMS) kurabilirim. O zaman fotoğrafları bana sormadan, tarayıcıdan kendiniz değiştirirsiniz. İsterseniz bunu hazırlayayım.

3. Alan adı yapısı (karar verildi)

  • dumansstudio.com → Bu tanıtım sitesi (halka açık ana sayfa)
  • ogrenci.dumansstudio.com → Öğrenci paneli (giriş)
  • yonetici.dumansstudio.com → Yönetici paneli

Sitedeki tüm “Öğrenci Girişi” butonları ogrenci.dumansstudio.com adresine, “Yönetici Girişi” butonu yonetici.dumansstudio.com adresine yönlendiriyor.

Yapmanız gereken: Öğrenci panelini, şu an bulunduğu dumansstudio.com adresinden ogrenci.dumansstudio.com alt adresine taşıyın (panelinizi barındıran firma yapabilir). Böylece dumansstudio.com bu tanıtım sitesine ayrılır.

4. Fotoğraf ipuçları (bana göndereceğiniz fotoğraflar için)

  • Galeri fotoğrafları: Dikey (portre) kareler en şık durur — telefonla çekilmiş, boyu eninden uzun fotoğraflar.
  • Boyut: Net ve iyi ışıklı olsun. Çok büyük dosyaları ben optimize ederim, siz orijinali gönderin.
  • Logo: Arka planı şeffaf (transparan) PNG olursa en temiz görünür.

Kısa özet

Yayınlamak: index.html dosyasını GitHub'a yükle → Netlify'ı bağla → Namecheap DNS. (Veya index.html'i doğrudan app.netlify.com/drop'a sürükle.)
Fotoğraf/yazı değiştirmek: bana söyle → yeni index.html'i sana vereyim → yükle.
Adresler: dumansstudio.com (site) · ogrenci.dumansstudio.com (öğrenci) · yonetici.dumansstudio.com (yönetici).
Takılırsan: bana yaz — her adımda ekran görüntüsüyle yardımcı olurum.
